# Customer Journey management

### Using Customer Journey on HIST System

#### Objective

This step-by-step guide outlines the steps to effectively use the Customer Journey feature in the HIST system for managing client therapy packages.

**1. Navigate to Customer Journey** [0:00](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=0)

![generated-image-at-00:00:00](https://loom.com/i/5da872fa60854e85ab98b5363907e176?workflows_screenshot=true)

* Open the HIST system.
* Locate and click on the 'Customer Journey' option.

&#x20;

**2. Start Therapy Session** [0:15](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=15)

![generated-image-at-00:00:15](https://loom.com/i/ea0336d946c24351aeedcc06ee008523?workflows_screenshot=true)

* Select the therapy package purchased by the client (e.g., 'loose fat and gain muscle set').
* The system will auto-fill relevant details such as:
  * Doctor
  * Dietitian
  * Salesperson (Health Secretary)
* Set the therapy start date (e.g., December 11, 2025).

&#x20;

**3. Confirm Therapy Details** [0:47](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=47)

![generated-image-at-00:00:47](https://loom.com/i/18e19e1e7ee542ffa6e9a50335bd326f?workflows_screenshot=true)

* Click 'Confirm' to finalize the therapy session.
* Review the timeline for therapy stages displayed as bubbles.

&#x20;

**4. Edit Therapy Information** [0:59](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=59)

![generated-image-at-00:00:59](https://loom.com/i/d6fff39adb36450a8a0b5b96a3066d61?workflows_screenshot=true)

* Access the therapy information drawer.
* Review the therapy details and navigate to the 'To-Do List'.
* Click 'Edit' to modify any dates or tasks as needed.

&#x20;

**5. Adjust Client Schedule** [1:21](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=81)

![generated-image-at-00:01:21](https://loom.com/i/e67cfbcdcf884bcbaa846874c87b0b66?workflows_screenshot=true)

* If a client needs to reschedule, update the date in the To-Do List (e.g., change week three to January 7, 2026).

&#x20;

**6. Manage Therapy Stages** [1:55](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=115)

![generated-image-at-00:01:55](https://loom.com/i/6dd4425d265a43e48236c8e83d9eea05?workflows_screenshot=true)

* Review preset templates for weeks 4, 5, and 6.
* Delete any unnecessary stages using the trash can icon.

&#x20;

**7. Add New Therapy Stage** [2:26](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=146)

![generated-image-at-00:02:26](https://loom.com/i/3abd0cccce5f4c3290b8325e98e26e42?workflows_screenshot=true)

* Click to add a new therapy stage at the end of the list.
* Select a date for the new stage (e.g., February 19, 2026).
* Specify the type of therapy (e.g., follow-up, IV therapy).

&#x20;

**8. Confirm New Stage Details** [3:00](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=180)

![generated-image-at-00:03:00](https://loom.com/i/95425dc62f404522b5ce23b97e083580?workflows_screenshot=true)

* Choose the appropriate session type (e.g., one session or 3+1 sessions).
* Click 'Confirm' to add the new stage.

&#x20;

**9. Track Client Attendance** [3:38](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=218)

![generated-image-at-00:03:38](https://loom.com/i/8c07f5ecf9984b28a36b215f654a3db3?workflows_screenshot=true)

* Check the checkbox for completed tasks to turn the bubble green.
* If tasks are overdue, the bubble will turn red as a reminder.

&#x20;

**10. Finish Therapy Session** [4:29](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=269)

![generated-image-at-00:04:29](https://loom.com/i/8ba509fa7e26499e8fe7282c7fa464ff?workflows_screenshot=true)

* Click 'Finish Therapy' to conclude the session.
* Optionally fill out results to document the therapy outcome.

&#x20;

**11. Review Therapy History** [5:02](https://loom.com/share/1e189b42202d4bb09e0dd5c3a504edb9?t=302)

![generated-image-at-00:05:02](https://loom.com/i/00ca521180114d7a85bfec7cc8c94635?workflows_screenshot=true)

* Access the therapy history for the client to view all completed sessions and details.

#### Cautionary Notes

* Ensure all client information is accurate before confirming therapy sessions.
* Regularly check for overdue tasks to maintain client engagement.

#### Tips for Efficiency

* Familiarize yourself with the therapy templates to speed up the setup process.
* Use the edit function proactively to accommodate client schedule changes.
